## Technologies Used
- [VSCode](https://code.visualstudio.com)
    - Code Editor
- [Git bash](https://gitforwindows.org)
    - Version control from windows
- [Heroku]()

- [Photoshop CS6](https://www.adobe.com/products/cs6.html)
